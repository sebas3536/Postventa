# Postventa MVP

Sistema de gestión postventa (reclamos y devoluciones) desarrollado como MVP usando tecnologías web básicas.

## Demo
ñ

## Tecnologías
- HTML5
- CSS3 (UI moderna)
- JavaScript
- JSON (simulación de base de datos)

## Funcionalidades
- Login de usuario
- Creación de reclamos
- Gestión de estados (Pendiente, En proceso, Resuelto)
- Dashboard con KPIs
- Interfaz moderna tipo aplicación

## KPIs
- Total de casos
- Casos pendientes
- Casos en proceso
- Casos resueltos

## Enfoque
Este proyecto simula un sistema real utilizando datos estáticos para validar la lógica de negocio antes de una implementación completa con backend.
